# **Introduction to Linux:**
Linux is a powerful and widely used operating system that is based on Unix. It is designed to manage computer hardware and software efficiently while providing a secure and stable environment. Linux is used in servers, personal computers, mobile devices, and cybersecurity systems. One of its biggest advantages is that it is flexible and customizable, which makes it suitable for beginners as well as advanced users like developers and ethical hackers.

## **What is Linux?**
Linux is an operating system (OS) that acts as a bridge between the user and the computer hardware. It controls system resources such as memory, processor, and storage, and allows users to run applications and manage files. Unlike some other operating systems, Linux is open-source, meaning its code is freely available for anyone to use, modify, and distribute. Popular versions of Linux include Ubuntu and Kali Linux.

---
## **Open Source vs Proprietary Operating System**

**Open Source OS:-**
An open-source operating system is one where the source code is publicly available. Anyone can view, modify, and distribute it according to their needs. Linux is a perfect example of an open-source OS. It is usually free to use, highly customizable, and supported by a large community of developers. This makes it more secure and flexible.

**Proprietary OS:-**
A proprietary operating system is owned by a company, and its source code is not available to the public. Users cannot modify it freely. Examples include Microsoft Windows and macOS. These systems are generally user-friendly and come with official support but offer less flexibility compared to Linux.

## **Absolute Path vs Relative Path**

**Absolute Path**
An absolute path is the complete path of a file or directory starting from the root (`/`). It always begins with `/` and does not depend on your current location.
 Example:  
`/home/kapil/Documents/file.txt`  
This path will always point to the same file no matter where you are.

**Relative Path**
A relative path is the path of a file or directory relative to your current working directory. It does not start with `/`.
Example:  
If you are in `/home/kapil`, then:  
`Documents/file.txt`
